# 🏨 Online Hotel Booking Analysis

## 📌 Project Overview

This project is a data analysis of an Online Hotel Booking system.  
The goal is to analyze booking patterns, customer behavior, and cancellation trends to gain useful business insights.

---

## 🎯 Objective

- Analyze hotel booking data
- Identify cancellation patterns
- Understand customer booking behavior
- Provide business insights to improve revenue

---

## 🛠️ Tools & Technologies Used

- Python 🐍
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Excel (Dataset)
- Jupyter Notebook

---

## 📊 Key Analysis Performed

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Cancellation Rate Analysis
- Monthly Booking Trends
- Customer Type Analysis
- Revenue Insights

---

## 📈 Key Insights

- Cancellation rate is higher in certain seasons
- Online bookings are more frequent than offline
- Certain customer types contribute more to revenue
- Peak booking months identified

---

## 📁 Project Files

- `hotel_booking.xlsx` → Raw dataset
- `analysis.ipynb` → Python analysis notebook
- `visualizations/` → Graphs and charts
